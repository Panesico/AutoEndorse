# Installation

## Arch / Debian / Ubuntu
``` shell
git clone https://github.com/Panesico/AutoEndorse
cd AutoEndorse
chmod +x install.sh
./install.sh
```

## Windows
``` powershell
git clone https://github.com/Panesico/AutoEndorse
cd AutoEndorse
.\install.ps1
```

# Usage

``` venv
python AutoEndorse.py
```

> [!WARNING]
> Remember to set the language to your web settings. Its not intended for log messages.
>
> be careful not to rm -rf ~ like I did :)
